
ConfUtils
=========

Утилиты для `Conf`.

Использование:

```
@@code file=_utils/ConfUtils#METHOD
```

confToText (метод)
------------------

Из файла `cfx` делает текст загруженного `Conf`.

Пример:

Исходный файл:

@@code file=attr1.cfx title=attr1.cfx

Используем:

```
@@code file=_utils/ConfUtils#confToText
    src=attr1.cfx 
```

получаем:

@@code file=_utils/ConfUtils#confToText
    src=attr1.cfx
