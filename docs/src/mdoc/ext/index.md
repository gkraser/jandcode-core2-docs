

jandcode-mdoc-ext (модуль)
==========================

Модуль определяет некоторые утилиты, которые используются при генерации документации
для проектов Jandcode. Можно использовать и в своих проектах,
если что то покажется полезным.

Подключение:

```groovy
include(MDocProject).with {
    depends.prod.add(
            "jandcode-mdoc-ext",
    )
}
```
